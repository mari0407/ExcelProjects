Attrition Dashboard Dashboard

Using an HR employee dataset from Kaggle, I created an Attrition Dashboard to provide insights of attrtion cases. For the dashboard I focused specifically on employees whose attrition was yes. In order to do this, I filtered the original dataset to include only the cases in which employee attrtion was yes. I then copied and pasted these cases to a new sheet to create the dashboard focused solely on attrtion cases. The dashboard showcases insights into the attrtion count based on job roles, travel freqeincy, distance from home, years of experience, attrtion count, and overall attrition percentage. I also chose to add gender, department, and overtime, as slicers to highlight how attrtion count deviates between these groups. The project reflect my ability to create a dashboard and use pivot tables.
